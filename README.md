# comp3011-search-engine
A simple search engine tool built for COMP3011 coursework, featuring web crawling, inverted indexing, and query processing.

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the program:
python src/main.py

## Commands

- build: crawl website and build index
- load: load index from file
- print <word>: show inverted index for a word
- find <query>: search pages containing query words
